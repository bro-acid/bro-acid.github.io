About this blog & Hello, world!

Acid, a graduate student majoring in Optical Engineering, once hoped to be an EE student. I like having fun with small electrical devices. During my undergraduate life, I learned how to use micro-controller unit, or MCU. Began with Intel 8051 from STC, then I learned STM32 and STM8 from ST Semiconductor and MSP430 from Texas Instruments. However, I didn’t get into these MCUs deep enough because the lacking of programming skills.

Last year, about November 11th, I bought a Raspberry Pi 3B. From that day on, I start to learn some linux commands. Actually, I have installed an Ubuntu on my old laptop three years ago but I didn’t use it frequently. It is hard for me to describe the feelings of interacting with a computer via terminal windows. Typing commands to control a computer is a primitive but reliable way. It’s really cool.

To improve my coding skills and leave a record of my study, I set this “blog” up. Actually this is my first homepage. It was cool if someone has an own homepage at the beginning of this century but now it seems to be an out-of-date thing. But today, here, I’d like to say:

Hello, world!
